Installation

This section guides you to install all required software and libraries to utilize Ekocart react for your projects fully


Node

Node.js® is a JavaScript run time built on Chrome’s V8 JavaScript engine. Node.js is required to build the Ekocart React. Download the latest version of Node and install it: Download Nodejs


NPM

NPM is the package manager for JavaScript and the world’s largest software registry. NPM is a separate project from Node.js and tends to update more frequently. As a result, even if you’ve just downloaded Node.js (and therefore npm), you’ll probably need to update your NPM .

npm install --global npm@latest

Verify that npm in successfully installed, and version of installed npm will appear

npm --version

React

Installation

1. First, unzip the zip file that you have download from ThemeForest. You will find a ekocart-react project and readme file.

2. Then after Extract zip File (ekocart-react.zip) into a folder, you have to work within.

3. Open a Node Prompt. Go to your Working Folder, And run the Following Command and wait for finish installation.

npm install

4. This command will install all the required node module into node_module/ inside your working folder.

5. You are ready to run the ekocart-react application for the first time

npm start

Deployment

Before the production, you must be changed in this Package.json file. In this file replace homepage attribute in your server URL.see the below screenshot.

Then after open node command prompt and go to the project directory. Type following command

npm run build

Now, Build is Completed and dist folder is created in your project root directory.Then After Directly Upload dist Folder in the root directory of your server.

Adding New Component

It’s easy to add your own custom component in this Ekocart react application if needs. Please follow the below steps for that.


Step 1 :

>> Please go to src->pages path and create a new folder.
>> Create an index.js file in this folder
>> Please refer below reference code or get a reference from the existing component which is already done in the project.

import React, { Component } from 'react';
class ComponentName extends Component {
  constructor(props) {
	super(props)
	this.state = {
	}
  }
  render() {
	return (
	  <div> 
	  </div>
	);
  }
}
export default ComponentName;

Step 2: Now you can create a function for getting data or calculate data. Please refer below code.

Function(Param){
	this.setState({
		..// Set State Value in Function
	})
	..// Function Logic
}

Step 3: Function can be called by following ways.

      3.1  If it needs to call function immediately after a component is mount then call the function as per below example.

componentDidMount() {
    ...
    this.FunctionName();
}

      3.2 If it is need to call function once the component render then call the function as per below example.

render() { 
  return( 
    <div> 
      ... 
      {this.FunctionName()}
    </div>
  ) 
}

      3.3 if it needs to call function with return data then it can be call by following way.

render()
{
   return(
       <div>
          ...
          <ComponentName functionreturn={this.FunctionName()} />
       </div>
       )
}

Redux

Redux is an open-source JavaScript library for managing application state. It is most commonly used with libraries such as React for building user interfaces.

Redux Thunk is a middleware that lets you call action creators that return a function instead of an action object. That function receives the store’s dispatch method, which is then used to dispatch regular synchronous actions inside the body of the function once the asynchronous operations have completed.
